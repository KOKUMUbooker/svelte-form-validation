## Initialize a sveltekit app
```bash
npx sv create svelte-form-validation --add tailwindcss ## Use defaults
```

## Move into project directory
```bash
cd svelte-form-validation
```

## Set up shadcn in the project
```bash
npx shadcn-svelte@latest init
```

## Add shadcn components
```bash
npx shadcn-svelte@latest add button
```
```bash
npx shadcn-svelte@latest add input
```
```bash
npx shadcn-svelte@latest add field
```
```bash
npx shadcn-svelte@latest add textarea
```
```bash
npx shadcn-svelte@latest add checkbox
```
